# io

## watch

```sh
cd docs/blog
gatsby develop
```

## deploy

```sh
cd docs/blog
yarn run deploy
```

## how to write posts

Suppose you are in `docs/blog`

- write a post in `content/posts/{{ post }}.md`
  - e.g., `readme.md`
- write a page in `content/pages/{{ page }}.md`
  - e.g., `about.md`
- add media in `static/media/{{ media }}.{png,jpg}`
  - e.g., `image-0.jpg`
