2025年ももう終わりますね #

TL;DRから書いて記事を書いているのだが、どういう意図で書いたかを自分でも忘れてわけがわからない。多分こんなことだろうと推測しながら書く。

いきなり重い話から。

東洋医学では女は7の倍数、男は8の倍数のときに体調に変わり目が訪れるらしい。1これを知ってからオリンピックの4年周期などどうでもよくなり8年周期が気になって仕方がなくなった。

ただし、これは身体起因の体調の話だ。8の倍数でもなんでもない年齢に、数年前になんかいい感じだったいい感じの人がなんか自殺してからこれまでなーんか体調がよくなかった。

常にニヤニヤしており、会社を辞めるときに胸中を打ち明けると

「お前、そんなに色々考えていたのか…」

と言われる程度にソシオパスルッキングかつソシオパス発言が多く(信頼できる人がいない→ソシオパス風→信頼できる人が現れない→…の無限サイクルが発生しているため一般人自認)、これもあまり共感はされなさそうだが今年ようやく落ち着いた気がした。いや、氣がした。

「病は氣から」と言うが、「全ては氣から」なのだ。先に結論がありそのための辻褄合わせに神話が存在するように、いい感じの人が死んだことにドラマ性を感じず、かと言って何も感じないことにしこりが残るような自分を正当化するための言い訳を見つけた「氣」が今年最大の収穫だった。

自分は文字通り天下国家を語るのが好きだ。ちょくちょくスケールのでかい話はするし、持株会で月1000円しか出資していない会社の経営方針に対して色々話す(話しても毛ほどの影響がないから話している無責任野郎かもしれない)。今までと異なり、今は理想から話すのではなく現実から話すようになった。前述の亡くなった人は希死念慮が消えなかった。その事実は少し前までうまく消化できないでいた。今までの人生で一番氣が合うと思った人なのに希死念慮という点を共有できなかった。自分は幸せなのに相手が死にたいと言っていると悲しいが相手も悲しいと言われても困る状況は、ニヤニヤしてしまいソシオパス扱いされる自分と同じだったかもしれないのに。

その人の死を克服するために自分が得た言い訳は「今を大切にする」ことだ。

「今を大切にする」というのは、言い換えるといかなるときも良いところを探す、自分を主語にできる部分だけで生きるということだ。希死念慮を持っているとかはどうでもよくて一緒にいて幸せなことに感謝をして、希死念慮があるという事実をあるがままに受け入れる。そういう風にすればよかったのかーというのを今更ながら気付いて、生きやすくなった気がする。

これまでは過去との整合性や理想との一貫性を考えていたが、現在は自分が今何をしていたのか、そのために何が必要なのかしか考えていない。今なら闇バイトだってできるだろう。

軽い話。

同様に会社員生活を経て「全ては現象だ」という原則も獲得した。今までも頭ではわかっていたのだが、実感がなかった。

「現象」というのは、人間の行動原理や感情を深く理解できなくても物事は前に進められる、という意味である。妄想めいた話かもしれないが、学校や世間ではしばしば「相手の気持ちを理解することが大事だ」と言われる。けれど実際には、他人の気持ちを完全に理解する必要はない。自分の目的に照らして相手がどう反応するのかを事実から学び、再現可能な形で扱えばよい。これが「全ては現象だ」という原則の要点だ。(人間や感情を特別扱いせず、現象として自分が得するように制御する感じ)

例えば、会社であるプロジェクトを実現したいとする。その場合、権限のない人に話しても状況は変わらない。必要なのは、決定権を持つ人に届く形で働きかけることだ。しかし、だからといって決定権者に直接話せば成功するとは限らない。上司を通さずに接触すると失敗することがある。

この失敗を「上司の気持ちを害したから」とか「社会では上司を介すべきだから」といった心理や規範で説明することもできる。だが、より実務的には「その会社がそういう構造で動いているから」と捉えるほうが正確で扱いやすい。つまり、世間一般に絶対の原則があるわけではなく、パターン化できる部分はあっても、組織や個人にはそれぞれ好む手順がある。うまくいかなかったら、それを道徳や感情の問題として解釈するより、「そのやり方では通らないのだ」と事実として受け止め、相手(会社)の流儀に合わせて通し方を学ぶ。私はそう理解するようになった。

こんなの基本的かもしれないと思われるかもしれないが、ADHDは実年齢の3分の2くらいだと思ってこいつは社会常識がないと理解してほしい。 by ADHD診断は受けていないし自分をADHDだと思っていないがよくお前はADHDだと言われるのでだったらそれを利用してやろうと思っている人

ADHDの人の年齢＝実際の年齢×２/３と考えてみてください。 ADHDの人の精神年齢は実際の年齢の２/３くらいに見積もるのが良いと考えます。つまり１０歳であっても精神年齢は６歳程度と見ます。ADHDの人は、行動の遂行に関わる脳の部位の皮質が厚くなるのに、普通の人より長く時間がかかると言われる報告もあります。ですから暦年齢通りに１０歳として扱い、期待すること自体に限界があります。2

仕事がまじでつまらん #

仕事がつまらない。最近はベーシックインカムだと思って生きている。色々あるが可能性として有力なのは次の通りである。

  1. 前提知識が足りない
  2. 尊敬できる同僚がいない
  3. 言われたことをやっているだけ
  4. 会話がない

なんかしらの対策をしないとなーと思って進めていることや今後進めようと思っていることがある。

知識を増やせ #

DRMの開発を担当しているが、セキュリティについてそんなに詳しくない。また会社のナレッジにもあまり精通しておらずミーティングに出てくる略語などを完全に理解しているわけではない。前提知識が欠けており自分の実力が十分に発揮されていないことが仕事がつまらない原因だ、という可能性がある。

セキュリティやネットワークに関する本を読んで、仕事が面白くならないかなーと思ってる。

環境を変えろ #

人と話すのが好きだ。特に賢くて自分と異なる意見の人と話すのが本当に好きだ。だが、そういう人を誰か一人上げろと言われても思い浮かばない。意見が異なるか異ならないか以前に一生浮いている気がする。加えて「こうなりたい」という人がチームや周りにいない。いや、正確にはいたが昇進して消えた。日常的に話す人間からの刺激がない。もっとこう、ビシバシ言われるか、何も言わないなら好きにさせてくれという感じ。

入社して1年経つとチームを変更することが可能なようなので、チーム変更に関して相談をするつもりだ。ただ、言い方が大事だ。今の同僚が「悪い」と思っているのではなく、単に「自分がなりたいとは思えない」というだけなのだが、言われて気分を害する人がいるのは学んできた。「業務を通じて今後伸ばしたいスキルが明確になり感謝している。今後はそれを日常的に学べる環境に移りたい。エンジニアではなく意思決定の背景を学べる場所で働きたい」などと言えば良いだろうか？

考えて働け #

仕事で脳みそを使わないから飽きた。使わなくても高給取りになれるという点ではよいのかもしれないが、ぶっちゃけ高給取りになりたいから入ったわけではないのでどうにかしたい。これはまぁ、隙間時間でインパクトを出せない俺が悪いのだ。

一番ルーティンワークではないかつ意思決定そのものなのが経営だと思ったので会社を始めることにした。業務委託はやってみたが、時間給なのが自分には合わなかった。日々が宿題に追われる感覚、会社も同じなのか気になる。失敗しても数百万円の借金が残る程度で、これで経営者に対する尊敬の念でも生まれて「雇われサイコー」となれば費用対効果は絶大だ。

会話しろ #

人と話すのが好きだ。(2度目)

もっと人と話したい。チームでリモートワークで働いている人が多く鬱だ。会社に来ても誰とも話さず帰るとベーシックインカムみが深い。年齢や性別に関わらず、刺激を受けて生きるのが好きなので、ホワイト企業は子供が生まれてから入るべきだったというのはちょっと思っている。チームでのランチや意見交換が少ない。

人の輪をつなげる&転職活動をする。周りの人を強制出社させるわけにはいかないのですでに来ている人と繋がるか、雰囲気の異なる会社に入る方が良い。PlayStation 6が出るまで辞めないと言ったがありゃ嘘だ。だが、転職をするというより自分に合いそうな会社をほんのり探っていきたい。出社マシマシ・コミュニケーションマシマシ(クライアントワークもやりたいです)・高学歴マシマシ・メリトクラシーマシマシな職場が良い。

最もドラスティックで最もオープンで最もセルフィッシュ #

今年も中々迷走していた。我を通せば軋轢が生まれるし、他人の意見を取り入れすぎれば己が消えるからだ。

俺は神ではないが、神もどきなのだ。

天皇海山群というものがある。日本付近の海山のことなのだが、基本的に天皇の名前が冠されている。しかしたまに「デトロイト海山」と言った、明らかにGHQに書き換えられた山川一問一答みたいな単語が出てくる。デトロイト海山は天皇ではないが、天皇海山群なのだ。天皇海山群がもし日本海山群だったらデトロイト海山の良さは消える。俺も我は出さないと良さは消えるが、我は通さなくても良い。間違えることもあるので手のひらを返しまくる、自分の間違いに気付かせてくれるそういう人とたくさん(仕事外で)会えたことは良かったと思う。

最もドラスティックに自分の意見を出し、最もオープンに反対意見を取り入れ、最もセルフィッシュに自分が得するような考えに至るのを来年は目標にしたいっすね。

来年もそんな感じで✊️
